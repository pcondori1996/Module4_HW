Language Quiz

Seconds Left : 30

Score: 0

To begin the game you must press the "Start" button. Once you have started the game you must answer as many questions as you can Before the timer runs out! The question will show a phrase in a language and you must pick the choose with the correct language on it. Every time you answer incorrectly the background changes to red and you lose 5 seconds

Question: Ex: Hola! Como estas hermosa?
